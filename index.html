My Projects

Agentic BI: Conversational Data Explorer

Agentic BI is a cutting-edge conversational Business Intelligence system that revolutionizes data interaction by allowing users to explore their datasets using natural language. This project leverages the power of **Large Language Models (LLMs)**, specifically **Mistral (via Ollama)**, orchestrated by **LangChain**, to automate complex data analysis tasks.

Key functionalities include:

  • Effortless Data Uploads: Users can easily upload their datasets in CSV format.
  • Natural Language Querying: Ask questions about your data using everyday English.
  • Automated SQL Generation: The system intelligently converts natural language questions into precise SQL queries.
  • Dynamic Data Visualization: Generates interactive charts (bar, line, scatter) with **Plotly Express** based on query results. A Chart Agent dynamically determines the best chart type.
  • AI-Powered Insights: An Insight Agent interprets query results and visualizations, providing concise, human-readable explanations of key trends, comparisons, and outliers, making data actionable for decision-makers.

The system architecture leverages a modular approach designed for scalability and ease of use, bridging the gap between raw data and actionable business intelligence through the innovative application of generative AI.

Tech Stack: Streamlit (Frontend), Python, LangChain, Mistral/Phi-2 (Ollama/Hugging Face), Plotly, CSV.
